 Personal Portfolio Website


The project demonstrates a complete portfolio website built from scratch using HTML5, CSS3, and JavaScript. It showcases my web design, responsive layout, and interactivity skills.


---

 Features
-  Fully responsive design for desktop and mobile  
-  Multi-page navigation (Home, About, Projects, Contact)  
-  Modern and clean UI with consistent theme  
-  Interactive contact form with basic validation  
-  Structured folder organization for scalability  

---

  Technologies Used
| Technology | Purpose |
|-------------|----------|
| **HTML5** | Website structure and content |
| **CSS3** | Styling, layout, and responsiveness |
| **JavaScript (ES6)** | Form validation and interactivity |
| **GitHub Pages** | Hosting the live demo |
